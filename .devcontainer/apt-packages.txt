# Linux system build dependencies for goscape-client.
#
# Native window/GL backend: github.com/go-gl/glfw/v3.3/glfw +
# github.com/go-gl/gl/v2.1/gl. Required at COMPILE time, not just runtime:
# pkg/jagex2/platform imports both, so even test binaries link against these
# native libraries. Authoritative source: go-gl/glfw build.go cgo LDFLAGS.
#
# Gio has been removed; the Wayland / EGL / Vulkan headers below predate that
# removal and may no longer be required — retained pending a dependency audit.
#
# libasound2-dev is for oto (audio).
#
# Single source of truth: consumed by .devcontainer/Dockerfile (the dev/CI
# image) and the Makefile `setup` target (host install). One package per line;
# blank lines and lines starting with '#' are ignored by both consumers.
#
# Package names target Debian trixie / recent Ubuntu. Vendor-neutral GL names
# are used throughout (libgl-dev, libgles-dev, libegl-dev) rather than the
# legacy Mesa-specific transitional names.
gcc
pkg-config
libffi-dev
libgl-dev
libwayland-dev
libx11-dev
libx11-xcb-dev
libxkbcommon-x11-dev
libxcursor-dev
libxrandr-dev
libxinerama-dev
libxi-dev
libxxf86vm-dev
libgles-dev
libegl-dev
libvulkan-dev
libasound2-dev
